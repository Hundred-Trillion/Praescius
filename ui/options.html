🚀 WELCOME TO PRAESCIUS

Praescius by Nanduri Labs is an advanced market observer Chrome Extension that intercepts live charts, parses trade data, compiles quantitative strategies using an in-memory compiler, and translates criteria in English directly into rules using Gemini AI.

1
Configure AI Translation Setup your Gemini API key in the Core & AI tab. This enables you to enter English descriptions like "Notify me when RSI exceeds 70 and close is in top 10%" and automatically translate them into compiled quantitative rules.
2
Connect Your Web3 Wallet Verify your developer profile or strategy signature using wallet integrations. Connect your wallet in the Portfolio Hub (Paper & Wallet) tab to sign verification payloads. This is a mock connector allowing instant personal signatures.
3
Run Simulations in Developer Panel Use the Developer Panel to load historical data (candles.jsonl) and simulate ticks. Watch the parser process binary payload tuples, monitor telemetry latencies, check cache-hits, and review console outputs in real-time.
4
Create Custom Indicators & Run Backtests Use the **Backtester & Builders** subtab in Portfolio Hub to write custom quantitative indicators, run parameter scan updates, view correlation metrics, review paper positions, and export trading logs to CSV.

💡 PROMPT TO RULE SYNTAX EXAMPLES

When creating custom notification rules on the extension sidebar, write clear English instructions. Here are some supported rules Gemini AI compiles:

  • "Notify me when RSI(14) is below 30 and price is near lower Bollinger Band" (Momentum Oversold check).
  • "Send an alert when there is a Bullish Engulfing pattern on high volume" (Candlestick formation check).
  • "Notify when MACD Hist is positive and EMA(9) crosses above EMA(21)" (Trend continuation check).
  • "Notify me when the lower wick ratio exceeds 60% and close is in top 10% of candle range" (Liquidity Sweep check).
