Quick Guide

1. Compile Rules: Add plain English instructions in the Dashboard rule-input. Gemini AI translates them to rules.
2. Import Strategies: Browse the ⚡ Library tab, click a card, and click Add to My Rules.
3. Options Centre: Click the gear icon (⚙) in the header to access advanced backtesting, scanners, and developer simulation engines.

Telegram Bot Setup

1. Create Bot: Message @BotFather on Telegram, send /newbot, and follow the steps. Copy the Bot Token provided.
2. Get Chat ID: Message your newly created bot to start a conversation. Then message @userinfobot to find your numerical User/Chat ID.
3. Connect: Paste both into the Dashboard settings. Praescius will automatically send trading alerts to your Telegram app.
